📝 ① 什么是字面量？

在 Python 中，字面量（Literal）指的是在代码中直接写出来的固定值。你可以把它理解成"所见即所得"的数据——你写的是什么，它就是什么。

💡
举个生活中的例子

假如你在作文本上写下"100"这个数字，那么这个 100 就是一个字面量——它本身就是 100，不代表别的意思。但如果你说"我的分数"，这就不是字面量，因为它不是一个固定的值，可能是 90 分，也可能是 95 分。

为什么要用字面量？

字面量是编程的基础。在写程序时，我们经常需要用到一些确定的值，比如：

🔢
数学计算

计算 1 + 1，这里的 1 就是数字字面量

💬
文本输出

打印"你好"，这里的"你好"就是字符串字面量

✅
逻辑判断

判断条件是否为真，True/False 就是布尔字面量

Python 
# 这里直接写出来的值都是字面量
print(100)           # 100 是整数字面量
print(3.14)          # 3.14 是浮点数字面量
print("你好")        # "你好" 是字符串字面量
print(True)          # True 是布尔字面量
🎨 ② 字面量的类型

Python 中有多种类型的字面量，我们来一一认识它们：

1️⃣ 数字类型字面量

数字类型的字面量分为整数和浮点数（小数）：

Python
# 整数字面量（没有小数点）
print(10)        # 正整数
print(-5)        # 负整数
print(0)         # 零

# 浮点数字面量（有小数点）
print(3.14)      # 正小数
print(-2.5)      # 负小数
print(0.0)       # 零的小数形式

# 科学计数法（对于很大或很小的数）
print(1.5e3)     # 1.5×10³ = 1500
print(2e-4)      # 2×10⁻⁴ = 0.0002
⚠️
注意

在 Python 中写数字时不能加逗号分隔符。比如一百万要写成 1000000 或 1_000_000（用下划线分隔更清晰），不能写成 1,000,000。

2️⃣ 字符串类型字面量

字符串就是一串文字，用引号包裹起来。可以用单引号 ' 或双引号 "，效果一样：

Python
# 单引号字符串
print('Hello')
print('我爱Python')

# 双引号字符串（和单引号效果一样）
print("Hello")
print("我爱Python")

# 三引号字符串（可以换行）
print("""这是第一行
这是第二行
这是第三行""")

# 空字符串
print("")         # 空字符串（什么也不输出）
🤔
单引号还是双引号？

在 Python 中，单引号和双引号没有区别。但如果字符串里本身包含引号，可以用另一种引号包裹：'他说："你好"' 或者 "I'm happy"。

3️⃣ 布尔类型字面量

布尔类型只有两个值：True（真）和 False（假），用来表示"是"或"否"：

Python
# 布尔字面量（注意首字母大写）
print(True)       # 真
print(False)      # 假

# 布尔值常用于判断
is_student = True
is_teacher = False

# 比较运算的结果也是布尔值
print(5 > 3)      # True
print(10 == 8)    # False
⚠️
重要提示

Python 中的布尔值 True 和 False 首字母必须大写！true 或 false（小写）会报错。

4️⃣ 特殊类型字面量

还有一个特殊的字面量 None，表示"什么都没有"或"空值"：

Python
# None 表示空值
result = None     # 初始化一个变量为空
print(None)       # 输出：None

# None 常用于表示"还没有值"
name = None       # 名字还没确定
age = None        # 年龄还没确定
📊 字面量类型总结
类型 说明 示例 用途
整数 没有小数点的数字 10, -5, 0 计数、索引、数学运算
浮点数 带小数点的数字 3.14, -2.5, 0.0 精确计算、科学计算
字符串 用引号包裹的文本 "你好", 'Python' 显示文字、存储信息
布尔 真或假 True, False 判断、条件、开关
None 空值 None 表示"没有值"
💬 ③ 什么是注释？

注释（Comment）是写在代码中的说明文字，专门给人看的，Python 解释器在运行时会完全忽略注释内容。

🌟
为什么需要注释？

想象一下，你写了一段复杂的代码，过了几个月后再看，可能连自己都看不懂了！注释就像是给代码写的"备忘录"，帮助你（和其他人）理解代码的意图和逻辑。

注释的作用 ✨

📖
解释代码

说明某段代码的作用和原理

🔍
标记重点

提醒自己或他人注意某些关键部分

🚫
临时禁用代码

测试时暂时"关闭"某些代码而不删除

📝
记录思路

写下解决问题的思路和想法

Python
# 这是一条注释，Python 不会执行它
print("Hello")  # 这条注释在代码后面

# 下面的代码被注释掉了，不会运行
# print("这句话不会输出")
🎯 ④ 注释的类型

1️⃣ 单行注释

单行注释以 # 号开头，从 # 到行尾的所有内容都是注释：

Python
# 这是一条单行注释
print("Hello, World!")

# 计算两个数的和
a = 10
b = 20
sum = a + b  # 这里也可以写注释

# 下面这行代码被注释掉了，不会运行
# print("这句话不会输出")

2️⃣ 多行注释

如果要写很长的注释，每行都加 # 会很麻烦。Python 提供了两种方式写多行注释：

方法一：连续使用多个单行注释

Python
# 这是第一行注释
# 这是第二行注释
# 这是第三行注释
print("程序开始运行")

方法二：使用三引号（推荐）

Python
"""
这是多行注释
可以写很多行
Python 会忽略这些内容
"""
print("程序开始运行")

'''
单引号的三引号也可以
效果是一样的
'''
print("继续执行")
💡
提示

三引号 """ 或 ''' 虽然常用来写多行注释，但严格来说它们是多行字符串。如果没有被赋值给变量，Python 会忽略它们，所以可以当作注释使用。

📌 注释的最佳实践

写好注释是一门艺术，下面是一些建议：

✅ 应该做的

  • 注释要简洁明了，一针见血
  • 解释为什么这样写，而不是怎么写的
  • 复杂的逻辑、算法一定要写注释
  • 对于不常见的技巧或解决方案要说明

❌ 不应该做的

  • 不要写显而易见的注释（如：# 这是一个变量）
  • 不要写过时的注释（代码改了，注释忘了更新）
  • 不要用注释来隐藏烂代码（应该重构代码）
  • 不要写太长的注释（会影响代码阅读）
Python
# ❌ 坏注释：太显而易见了
x = 10  # 把 10 赋值给 x

# ✅ 好注释：解释了为什么
x = 10  # 班级最多容纳 10 个学生

# ❌ 坏注释：没用的信息
# 这是一个 for 循环
for i in range(5):
    print(i)

# ✅ 好注释：解释了目的
# 遍历所有学生，打印他们的序号
for i in range(5):
    print(i)
🎨 综合示例

下面是一个综合运用字面量和注释的例子：

Python
"""
学生信息管理程序
作者：小明
日期：2024-01-15
"""

# 学生基本信息（使用不同类型的字面量）
name = "张三"           # 字符串字面量
age = 14                # 整数字面量
height = 1.65           # 浮点数字面量
is_monitor = True       # 布尔字面量
phone = None            # None 字面量（电话号码待填写）

# 输出学生信息
print("===== 学生信息 =====")
print("姓名:", name)
print("年龄:", age, "岁")
print("身高:", height, "米")
print("是否是班长:", is_monitor)

# 计算 5 年后的年龄
future_age = age + 5
print(f"{name} 五年后的年龄是 {future_age} 岁")
