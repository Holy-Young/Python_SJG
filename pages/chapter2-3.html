1. Python 的基本数据类型

在 Python 中，每个数据都有自己的类型。就像生活中的物品分为不同类别一样，数字、文字、真假值在程序中也有不同的分类。Python 有四种最基础的数据类型。

🔢
整数 (int)
整数就是没有小数部分的数字，可以是正数、负数或零。用于计数、排序等场景。
42, -17, 0, 999, -100
📊
浮点数 (float)
浮点数就是带小数点的数字，用于表示更精确的数值，比如身高、体重、温度等。
3.14, -0.5, 2.0, 99.99
📝
字符串 (str)
字符串就是文本，由一串字符组成。必须用引号（单引号或双引号）括起来。
"Hello", '你好', "123", 'Python'
✅
布尔值 (bool)
布尔值只有两个值：True（真）和 False（假），用于判断条件是否成立。
True, False

💡 查看数据类型

使用 type() 函数可以查看任何数据的类型：

Python 
# 查看不同数据的类型
print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type("Hello"))     # <class 'str'>
print(type(True))        # <class 'bool'>

# 变量也可以查看类型
age = 14
name = "小明"
print(type(age))         # <class 'int'>
print(type(name))        # <class 'str'>
💡
重要提示

字符串中的数字不是数字类型！比如 "123" 是字符串，不能直接参与数学运算。想要运算必须先转换类型。

2. 数据类型转换

有时候我们需要把一种数据类型转换成另一种。比如把字符串 "123" 转成整数 123，或者把整数 100 转成字符串 "100"。Python 提供了四个常用的转换函数。

🔄 类型转换示例

原始值
"123"
str 类型
→
int("123")
123
int 类型
原始值
3.14
float 类型
→
int(3.14)
3
int 类型
原始值
100
int 类型
→
str(100)
"100"
str 类型

📝 四种转换函数

Python 
# 1. int() - 转换为整数
num1 = int("123")      # "123" → 123
num2 = int(3.14)       # 3.14 → 3 (小数部分会被舍去)
num3 = int(3.99)       # 3.99 → 3 (不是四舍五入!)

# 2. float() - 转换为浮点数
num4 = float("3.14")   # "3.14" → 3.14
num5 = float(100)      # 100 → 100.0
num6 = float("99")     # "99" → 99.0

# 3. str() - 转换为字符串
text1 = str(123)       # 123 → "123"
text2 = str(3.14)      # 3.14 → "3.14"
text3 = str(True)      # True → "True"

# 4. bool() - 转换为布尔值
b1 = bool(1)           # 1 → True (非零数字都是 True)
b2 = bool(0)           # 0 → False
b3 = bool("hello")     # "hello" → True (非空字符串是 True)
b4 = bool("")          # "" → False (空字符串是 False)
⚠️
常见错误

不能把不合法的字符串转成数字！比如 int("hello") 会报错，因为 "hello" 不是数字。

🎯 实际应用场景

Python 
# 场景1: 用户输入的都是字符串，需要转换
age_str = input("请输入你的年龄: ")  # 用户输入 "14"
age = int(age_str)                   # 转换成整数 14
print("你明年" + str(age + 1) + "岁") # 需要把整数转回字符串才能拼接

# 场景2: 计算平均分
score1 = "85"
score2 = "92"
score3 = "78"
average = (int(score1) + int(score2) + int(score3)) / 3
print(f"平均分: {average}")

# 场景3: 价格计算
price = "99.99"
quantity = 3
total = float(price) * quantity  # 先把字符串转成浮点数
print(f"总价: {total}元")

3. 算术运算符

算术运算符用于进行数学计算。Python 提供了丰富的运算符，除了基本的加减乘除，还有一些特殊的运算。

运算符 名称 示例 结果
+ 加法 10 + 3 13
- 减法 10 - 3 7
* 乘法 10 * 3 30
/ 除法 10 / 3 3.333...
// 整除（向下取整） 10 // 3 3
% 取余（取模） 10 % 3 1
** 幂运算 2 ** 3 8 (2的3次方)

🔍 重点理解：整除和取余

Python 
# 除法 / vs 整除 //
print(10 / 3)    # 3.3333333333333335 (保留小数)
print(10 // 3)   # 3 (只要整数部分)

# 取余 % 的应用
print(10 % 3)    # 1 (10除以3余1)
print(17 % 5)    # 2 (17除以5余2)

# 判断奇偶数
num = 7
if num % 2 == 0:
    print("偶数")
else:
    print("奇数")   # 输出: 奇数

# 幂运算 **
print(2 ** 3)    # 8 (2³ = 2×2×2)
print(3 ** 2)    # 9 (3² = 3×3)
print(10 ** 0)   # 1 (任何数的0次方都是1)

🎨 字符串的特殊运算

+ 和 * 运算符对字符串也有特殊作用：

字符串拼接（+）
"Hello" + " " + "World"
→ "Hello World"
字符串重复（*）
"Python" * 3
→ "PythonPythonPython"
实际应用
"=" * 20
→ "====================" (生成分割线)
Python 
# 字符串拼接
first_name = "张"
last_name = "三"
full_name = first_name + last_name  # "张三"

# 字符串重复
border = "=" * 30
print(border)
print("欢迎使用Python")
print(border)
# 输出:
# ==============================
# 欢迎使用Python
# ==============================

# 注意: 字符串不能和数字直接相加
age = 14
# message = "我" + age + "岁"  # 错误! 会报错
message = "我" + str(age) + "岁"  # 正确! 先转成字符串

4. 比较运算符

比较运算符用于比较两个值的大小关系，结果永远是布尔值（True 或 False）。

运算符 名称 示例 结果
== 等于 5 == 5 True
!= 不等于 5 != 3 True
> 大于 5 > 3 True
< 小于 5 < 3 False
>= 大于等于 5 >= 5 True
<= 小于等于 3 <= 5 True
⚠️
注意区分

= 是赋值（把右边的值给左边），== 是比较（判断左右是否相等）。这是新手最容易搞混的地方！

Python 
# 数字比较
print(10 > 5)        # True
print(10 == 10)      # True
print(10 != 5)       # True
print(3 >= 5)        # False

# 字符串也可以比较 (按字典序)
print("apple" < "banana")   # True
print("abc" == "abc")       # True
print("Python" != "Java")   # True

# 实际应用: 判断成绩等级
score = 85
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")    # 输出这个
else:
    print("不及格")

# 判断年龄段
age = 14
is_teenager = age >= 13 and age <= 19  # True
print(f"是青少年吗? {is_teenager}")

5. 逻辑运算符

逻辑运算符用于组合多个条件判断。Python 有三个逻辑运算符：and（与）、or（或）、not（非）。

运算符 名称 说明 示例
and 与 两个条件都为真才是真 True and True → True
or 或 至少一个条件为真就是真 True or False → True
not 非 取反，真变假，假变真 not True → False

Python 
# and - 两个条件都要满足
age = 14
has_id = True
can_enter = age >= 13 and has_id  # True (年龄够了，也有证件)

# or - 满足其中一个条件就行
is_weekend = False
is_holiday = True
can_rest = is_weekend or is_holiday  # True (虽然不是周末，但是假期)

# not - 取反
is_raining = False
can_play_outside = not is_raining  # True (没下雨，可以出去玩)

# 组合使用
score = 85
attitude = "good"
can_get_award = score >= 80 and attitude == "good"  # True
print(f"能获奖吗? {can_get_award}")

# 实际场景: 判断能否参加活动
age = 14
has_permission = True
is_member = False

# 必须满足: (年龄>=13 且有家长同意) 或者 是会员
can_join = (age >= 13 and has_permission) or is_member
print(f"可以参加吗? {can_join}")  # True
💡
记忆技巧

and 就像"又...又..."，两个都要满足；or 就像"或者"，满足一个就行；not 就是"不"，把结果反过来。

6. 运算符优先级

当一个表达式中有多个运算符时，Python 会按照优先级顺序计算。就像数学中"先乘除，后加减"一样。

1
( )
括号优先级最高，先算括号里的
2
**
幂运算
3
* / // %
乘法、除法、整除、取余
4
+ -
加法、减法
5
> < >= <= == !=
比较运算符
6
not
逻辑非
7
and
逻辑与
8
or
逻辑或（优先级最低）
Python 
# 示例1: 先乘除，后加减
result = 10 + 5 * 2     # 先算 5*2=10，再算 10+10=20
print(result)           # 20

# 示例2: 括号优先
result = (10 + 5) * 2   # 先算括号 10+5=15，再算 15*2=30
print(result)           # 30

# 示例3: 幂运算优先级高
result = 2 * 3 ** 2     # 先算 3²=9，再算 2*9=18
print(result)           # 18

# 示例4: 比较运算在算术之后
age = 14
result = age + 1 > 13   # 先算 age+1=15，再比较 15>13，结果True
print(result)           # True

# 示例5: 逻辑运算最后算
result = 10 > 5 and 20 < 30  # 先算比较，再算逻辑
# 相当于: True and True → True
print(result)           # True

# 复杂示例
age = 14
score = 85
result = age >= 13 and score > 60 or score == 100
# 计算顺序:
# 1. age >= 13 → True
# 2. score > 60 → True
# 3. score == 100 → False
# 4. True and True → True
# 5. True or False → True
print(result)           # True
💡
建议

如果不确定优先级，就用括号！括号可以让代码更清晰，也不容易出错。比如 (age >= 13) and (score > 60) 比 age >= 13 and score > 60 更容易理解。

# 练习3: 购物结算系统
