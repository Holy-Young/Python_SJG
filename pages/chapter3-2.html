① 为什么需要循环？

想象一下：你要打印 "加油！" 这句话 100 次。如果不用循环，你需要写 100 行 print("加油！")，不仅超级麻烦，代码改起来也很痛苦。

❌ 不用循环：重复写代码
print("加油！") # 第1次 print("加油！") # 第2次 print("加油！") # 第3次 # ... 还要写 96 行 print("加油！") # 第100次

😫 太麻烦了！

✅ 用 while 循环：简洁优雅
count = 1 while count <= 100: print("加油！") count += 1 # 4 行搞定！

😎 4 行代码搞定！

💡
生活中的循环例子

闹钟每天重复响；游戏里角色不停走路；ATM 机反复等待你输入密码……这些都是"循环"在现实中的体现。while 循环就是让程序能够重复执行一段代码，直到某个条件不再成立为止。

② while 基本语法与执行流程

📐 基本语法格式

语法格式 
while 条件:
    # 条件为 True 时，反复执行这里的代码
    执行的代码A
    执行的代码B
    # 通常这里要让条件"往假的方向变化"，否则永远不会停

后续代码  # 条件变为 False 后，才执行这里

🗺️ 执行流程图

while 循环的执行顺序如下图所示：每次进入循环前都先判断条件，条件为真才执行，执行完再回头重新判断，形成一个"圈"。

开始
↓
判断条件
✓ 条件成立
↓
执行循环里的代码
↩ 回头重新判断
✗ 条件不成立
↓
跳出循环，继续往后

💡 记住这个"圈"：判断 → 执行 → 回头判断 → 执行 → … → 条件不成立 → 结束

🔢 计数器模式——最常用的写法

用一个变量来计数，每次循环让它加 1，当它超过目标值时停止。这是 while 循环最经典的用法：

Python 
# 打印 1 到 5
count = 1           # ① 初始化计数器
while count <= 5:   # ② 判断条件
    print(count)    # ③ 执行操作
    count += 1      # ④ 更新计数器（关键！别忘了这行）

# 输出：1  2  3  4  5

来看一个更有趣的例子——用循环算 1 到 100 的总和：

Python 
total = 0
i = 1
while i <= 100:
    total += i   # 等价于 total = total + i
    i += 1

print(f"1 到 100 的总和是：{total}")
# 输出：1 到 100 的总和是：5050

写 while 循环记住一件事：让条件最终能变假

只要循环体里有让变量"往假的方向变化"的代码（比如 count += 1），循环就一定能停下来。如果程序真的转个不停也别慌——后面我们会教你怎么轻松处理这种情况 😊

③ break — 提前结束循环

有时候，我们不想等到条件变假才退出，而是在某个特殊情况下立刻停止循环。这就是 break 的用途。 break 一执行，循环立刻结束，程序跳到循环后面继续执行。

Python 
# 找到第一个能被 7 整除的数
i = 1
while i <= 100:
    if i % 7 == 0:
        print(f"找到了！{i} 能被 7 整除")
        break        # 找到就立刻停止，不再继续
    i += 1

print("循环结束")
# 输出：找到了！7 能被 7 整除
#       循环结束

🎮 猜数字小游戏（break 实战）

猜数字游戏是 break 最经典的应用：反复让用户猜，猜对了就 break 退出。来试试这个互动模拟器：

🎯 猜数字游戏模拟器
程序随机选了 1~20 的一个数，你有 6 次机会猜中它！
剩余次数：6 / 6
系统：已选好一个数，开始猜吧！

上面的游戏，对应的 Python 代码逻辑是这样的：

Python 
import random
secret = random.randint(1, 20)  # 随机选 1~20 的数
attempts = 0
max_attempts = 6

while attempts < max_attempts:
    guess = int(input("请猜一个 1~20 的数："))
    attempts += 1

    if guess == secret:
        print(f"🎉 恭喜猜对了！答案就是 {secret}，共猜了 {attempts} 次")
        break              # 猜对了，立刻退出循环
    elif guess < secret:
        print("太小了，再大一点 ↑")
    else:
        print("太大了，再小一点 ↓")

    if attempts == max_attempts:
        print(f"😢 机会用完了！答案是 {secret}")

④ continue — 跳过本次，继续下一次

continue 和 break 不同：break 是整个退出循环，而 continue 只是跳过当次剩余代码，直接回到条件判断，继续下一次循环。

🎒 用生活场景记住区别
🛑 break = 老师突然说"下课！全班立刻收拾书包走人"——整个循环直接结束
⏭️ continue = 做卷子时遇到不会的题，"这题跳过！" ——跳过这道题，但还要继续做后面的题
🛑 break
遇到 break，整个循环立刻终止，跳出去执行循环后面的代码。 i = 1 while i <= 5: if i == 3: break # 到 3 就停 print(i) i += 1 # 输出：1 2
⏭️ continue
遇到 continue，跳过本次循环剩余代码，回头重新判断条件，继续下一次。 i = 0 while i < 5: i += 1 if i == 3: continue # 跳过 3 print(i) # 输出：1 2 4 5

🍬 continue 实战：发糖果（跳过感冒的同学）

# 给 1~5 号同学发糖果，但 3 号感冒了，跳过他
i = 0
while i < 5:
    i += 1
    if i == 3:
        print(f"  {i} 号感冒了，跳过！")
        continue      # 跳过 3 号，继续发给后面的同学

# 给 4 号同学发糖果 🍬
# 给 5 号同学发糖果 🍬

⑤ 一直转的循环：while True

把 while 的条件直接写成 True，因为 True 永远成立，循环就会一直转下去——这叫持续循环（有时也叫"无限循环"）。听起来很危险？其实只要配合 break，它就是一个非常好用的结构！

Python 
# 反复让用户输入密码，直到输入正确才停
while True:
    password = input("请输入密码：")
    if password == "abc123":
        print("✅ 密码正确，欢迎进入！")
        break        # 猜对了！break 让循环停下来
    else:
        print("❌ 密码错误，再试一次～")
💡
什么时候用 while True？

当你一开始不知道循环要跑几次，只知道"满足某个条件就停"时，用 while True + break 非常方便。比如：反复让用户输入直到输入合法；游戏里一直等玩家操作直到玩家退出。

😊
程序一直转停不下来？别慌，这很正常！
就像玩"老鹰捉小鸡"游戏，老师喊"停"游戏就结束。程序也有紧急停止方法——在终端里按 Ctrl + C，它就会乖乖停下来，不会伤到电脑。
养成一个小习惯：写完循环后问自己"这个循环最终能停吗？" ✨

⑥ while 嵌套循环
🎯
单层 while 已经掌握了吗？来挑战双层！ 嵌套循环是 while 里稍微难一点的知识，不用担心——我们先从超简单的生活例子开始，一步一步来 😊

循环里面还可以再套一个循环，这就叫嵌套循环。外层循环每执行一次，内层循环就会完整地跑完一遍。

🔍 结构理解

🔵 外层 while（控制"第几组"）
外层每循环一次，内层就从头跑一遍
🟣 内层 while（控制"组内每一项"）
内层执行完一整圈后，控制权回到外层，外层再走一步
外层第 1 次 → 内层跑完 1 圈
外层第 2 次 → 内层再跑完 1 圈
外层第 3 次 → 内层再跑完 1 圈
… 以此类推

🕐 第一个例子：刷牙计划（超简单）

先用一个生活化的小例子感受一下嵌套循环的节奏：外层控制"今天星期几"，内层控制"每天刷几次牙"。

Python 
# 连续 3 天，每天刷 2 次牙
day = 1
while day <= 3:            # 外层：今天是第几天
    print(f"第 {day} 天：")
    time = 1
    while time <= 2:       # 内层：今天刷第几次
        print(f"  刷牙！（第 {time} 次）")
        time += 1
    day += 1

# 输出：
# 第 1 天：
#   刷牙！（第 1 次）
#   刷牙！（第 2 次）
# 第 2 天：
#   刷牙！（第 1 次）
#   刷牙！（第 2 次）
# 第 3 天：
#   刷牙！（第 1 次）
#   刷牙！（第 2 次）
🔑
嵌套循环的关键点：内层变量要重置！

注意看上面代码：每次外层循环开始时，都要把内层的变量（time）重新设为 1。如果忘记了，内层循环从第 2 天起就不会执行了（因为 time 已经是 3 了）。

📐 基本语法结构

语法格式 
i = 1
while i <= 外层次数:       # 外层循环
    j = 1                  # ← 每次外层循环开始，内层变量要重置！
    while j <= 内层次数:   # 内层循环
        # 执行操作
        j += 1
    i += 1

🔢 再进一步：打印星号矩阵

Python 
# 打印 3 行 × 4 列的星号
row = 1
while row <= 3:       # 外层：控制行数
    col = 1
    line = ""
    while col <= 4:   # 内层：控制每行的列数
        line += "★ "
        col += 1
    print(line)
    row += 1

# 输出：
# ★ ★ ★ ★
# ★ ★ ★ ★
# ★ ★ ★ ★

🏆 进阶挑战（选做）：九九乘法表

⭐ 这是一道挑战题！星号矩阵已经看懂了就来试试这个经典例子；还没完全明白也没关系，先巩固上面的内容再回来 😊

九九乘法表是嵌套循环最经典的例子：外层控制"第几行（被乘数）"，内层控制"第几列（乘数）"。

Python 
i = 1
while i <= 9:           # 外层：被乘数 1~9
    j = 1
    while j <= i:       # 内层：乘数 1~i（不超过被乘数）
        result = i * j
        print(f"{j}×{i}={result}", end="\t")  # end="\t" 表示不换行，用制表符隔开
        j += 1
    print()             # 每行结束后换行
    i += 1

⚡
嵌套循环的执行次数 = 外层次数 × 内层次数

比如九九乘法表：外层 9 次 × 内层平均约 5 次 ≈ 共执行约 45 次乘法。嵌套层数越多，执行次数增长越快，所以一般不建议超过三层嵌套。
