🐍 ① Python 是什么？

Python 是著名的"龟叔" Guido van Rossum 在 1989 年圣诞节期间，为了打发无聊的假期而编写的一个编程语言。牛人就是牛人，打发无聊竟然写出了这么厉害的东西！

🌟
名字的由来

Python 这个名字来自英国喜剧团体"蒙提·派森"（Monty Python），和蛇没有直接关系——尽管它的 Logo 是条蟒蛇。创始人 Guido 很喜欢这个喜剧团体，就用了这个名字。

Python 的优点 👍

✅
语法简单易学

代码接近自然语言，初学者很容易上手

✅
功能强大

海量第三方库，AI、爬虫、数据分析一站搞定

✅
跨平台

Windows、Mac、Linux 都能运行

✅
社区活跃

全球数百万开发者，遇到问题很容易找到帮助

Python 的缺点 👎

⚠️
运行速度较慢：Python 是解释型语言，执行时需逐行翻译，比 C/C++ 慢
⚠️
代码不能加密：发布程序等于公开源代码，商业保密有一定困难

⬇️ ② 安装 Python

在开始写代码之前，我们需要在电脑上安装 Python。步骤非常简单：

1
打开官网

访问 https://www.python.org/downloads/，点击黄色大按钮下载最新版本

2
运行安装程序

双击下载的 .exe 文件，务必勾选 "Add Python to PATH"，然后点击 Install Now

3
验证安装成功

按 Win + R，输入 cmd 打开命令行，输入 python --version，能看到版本号就成功了！

🎯
重要提示

安装时一定要勾选 Add Python to PATH！如果忘了勾选，之后在命令行输入 python 会提示"找不到命令"，需要重新安装。

👋 ③ 第一个程序：Hello World

按照编程界的传统，学任何语言都要先写一个打印 "Hello, World!" 的程序：

hello.py 
# 这是我的第一个 Python 程序
# "#" 开头的是注释，电脑会忽略这一行

print("Hello, World!")
print("你好，Python！")
print("我是一名初中生，我在学编程！")
运行结果
Hello, World!
你好，Python！
我是一名初中生，我在学编程！
💬
什么是注释？

用 # 开头的内容叫"注释"，电脑运行时完全忽略它。注释是写给人看的，用来解释代码的作用。养成写注释的好习惯非常重要！

🖨️ ④ print 函数详解

print() 是 Python 中最常用的函数之一，用来把内容输出到屏幕。让我们深入了解它的各种用法：

基本用法

Python
print("你好")      # 打印字符串
print(123)         # 打印数字
print(3.14)        # 打印小数
print(True)        # 打印布尔值
运行结果
你好
123
3.14
True

一次打印多个值

print() 括号里可以放多个值，用逗号隔开，默认用空格分隔：

Python
print("我叫", "小明", "今年", 13, "岁")
运行结果
我叫 小明 今年 13 岁

sep 参数：自定义分隔符

Python
print("2024", "01", "01", sep="-")      # 用 - 分隔
print("www", "python", "org", sep=".")  # 用 . 分隔
print("A", "B", "C", sep="")           # 无分隔符
运行结果
2024-01-01
www.python.org
ABC

end 参数：控制结尾

end 控制 print 打印完后结尾加什么（默认是换行 \n）：

Python
print("苹果", end="、")
print("香蕉", end="、")
print("橙子")
运行结果
苹果、香蕉、橙子
🎯
小结 print() 的参数

sep 控制多个值之间的分隔符（默认空格）；end 控制打印结束后的字符（默认换行）。
